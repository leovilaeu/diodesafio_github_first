## Link GIT: https://git-scm.com/downloads
## Comandos básicos:
ls  - Lista diretório
mkdir - Cria Diretório
cd - Caminha até diretório
cd .. - Retorna um nível diretório



## Ações básicas GIT:

*git init* - Inicia repositório GIT na pasta selecionada.

*git add*  
*git commit* 
*git clone*  
*git push* 
*git pull* 


## Flags

*-a (ex: ls -a)* - Lista diretórios ocultos.
